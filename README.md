# Kiosk_ljb_Spring9_ch2
* Spring9기 ch2 키오스크 과제

---
# 📌프로젝트 소개
사용자가 입력하여 메뉴를 선택하고 해당 메뉴 아이템을 골라 장바구니에 담아서
사용자의 유형에 따라 총 금액에서 할인되어서 출력하는 간단한 콘솔 기반 키오스크 프로그램입니다.
